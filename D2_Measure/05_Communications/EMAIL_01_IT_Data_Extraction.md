# Data Extraction Request

## English
**Subject:** Data Extraction Request — 6S Project (Measure Phase)

**Dear IT Team,**

To establish the baseline for our 6S project, we request an extraction of production data from the ERP system for the last 6 months (Jan 01 - Jun 30).

**Required Data Fields:**
- Date and Shift
- Total Production Volume (by SKU)
- Downtime Duration (minutes) and Reason Code
- Defect Counts (specifically Capping Defects)

Please provide this in Excel format by July 10, 2026. This data is critical to accurately measure our current performance gap.

Thank you,
