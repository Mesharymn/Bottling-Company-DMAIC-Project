# 01 — Project Charter | وثيقة المشروع

This folder contains the foundational Define-stage documents that establish the project's purpose, boundaries, and direction.
يحتوي هذا المجلد على الوثائق الأساسية لمرحلة التحديد التي تُحدد غرض المشروع وحدوده واتجاهه.

| File / الملف | Description / الوصف |
|---|---|
| `Project_Charter.md` | Business case, SMART goals, team, and scope / وثيقة المشروع، الأهداف الذكية، الفريق، والنطاق |
| `Problem_Statement.md` | Detailed description with current vs. target data and financial/operational impact / وصف تفصيلي مع مقارنة الأداء الحالي بالمستهدف والتأثير المالي والتشغيلي |
