# 02 — Process Mapping

This folder contains multiple SIPOC documents for the Water Bottling Company. The first SIPOC provides an end-to-end overview of the entire production line. Each subsequent SIPOC zooms into a specific sub-process to provide deeper clarity on inputs, outputs, and responsible parties at each stage.

| File | Scope |
|---|---|
| `SIPOC_00_Overview.md` | End-to-end production line overview |
| `SIPOC_01_Water_Treatment.md` | Water treatment and purification process |
| `SIPOC_02_Bottle_Blowing.md` | Preform heating and bottle blowing process |
| `SIPOC_03_Filling.md` | Bottle filling and volume control process |
| `SIPOC_04_Capping.md` | Capping and seal integrity process |
| `SIPOC_05_Labeling.md` | Label application and alignment process |
| `SIPOC_06_Packaging.md` | Packaging, palletizing, and dispatch process |
