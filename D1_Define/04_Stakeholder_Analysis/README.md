# 04 — Stakeholder Analysis | تحليل أصحاب المصلحة

This folder contains the complete stakeholder analysis across five dimensions.
يحتوي هذا المجلد على تحليل أصحاب المصلحة الكامل عبر خمسة محاور.

| File / الملف | Description / الوصف |
|---|---|
| `Stakeholder_Identification.md` | Who the stakeholders are, their type, role, and area of influence / من هم أصحاب المصلحة، نوعهم، دورهم، ومجال تأثيرهم |
| `Stakeholder_Mapping.md` | Power/Interest/Impact matrix, Stakeholder vs CTQ, and Influence Map / مصفوفة القوة/الاهتمام/التأثير، ومقارنة أصحاب المصلحة بمتطلبات الجودة الحرجة، وخريطة التأثير |
| `Engagement_and_Risk.md` | Engagement strategy per stakeholder and risk register / استراتيجية التفاعل لكل صاحب مصلحة وسجل المخاطر |
| `Key_Questions_Summary.md` | Strategic summary answering the 9 critical stakeholder questions / ملخص استراتيجي يُجيب على الأسئلة التسعة الحرجة لأصحاب المصلحة |
