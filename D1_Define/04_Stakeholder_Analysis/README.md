# 04 — Stakeholder Analysis

This folder contains the complete stakeholder analysis across five dimensions.

| File | Description |
|---|---|
| `Stakeholder_Identification.md` | Who the stakeholders are, their type, role, and area of influence |
| `Stakeholder_Mapping.md` | Power/Interest/Impact matrix, Stakeholder vs CTQ, and Influence Map |
| `Engagement_and_Risk.md` | Engagement strategy per stakeholder and risk register |
| `Key_Questions_Summary.md` | Strategic summary answering the 9 critical stakeholder questions |
