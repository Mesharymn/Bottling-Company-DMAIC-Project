# 03 — Voice of Customer | صوت العميل

This folder translates raw customer and stakeholder feedback into measurable CTQ requirements.
يُترجم هذا المجلد ملاحظات العملاء وأصحاب المصلحة الخام إلى متطلبات جودة حرجة (CTQ) قابلة للقياس.

| File / الملف | Description / الوصف |
|---|---|
| `VOC_Translation.md` | Raw VOC from external customers and internal sales with root-cause reasoning / صوت العميل الخام من العملاء الخارجيين والمبيعات الداخلية مع تحليل السبب الجذري |
| `VOC_to_CTQ.md` | Structured translation: VOC → Need → CTQ → Metric → Target / الترجمة المنظمة: صوت العميل ← الحاجة ← متطلب الجودة الحرج ← المقياس ← الهدف |
