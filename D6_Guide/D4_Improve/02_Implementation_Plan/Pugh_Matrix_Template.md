# Pugh Matrix Template

## English
**Problem:** [Name]
**Baseline Solution:** [Current process or standard solution]

| Criteria | Weight (1-5) | Baseline | Solution A | Solution B | Solution C |
|----------|--------------|----------|------------|------------|------------|
| [Criterion 1, e.g., Cost] | [Weight] | S | [+ / - / S] | [+ / - / S] | [+ / - / S] |
| [Criterion 2, e.g., Speed] | [Weight] | S | [+ / - / S] | [+ / - / S] | [+ / - / S] |
| [Criterion 3, e.g., Quality] | [Weight] | S | [+ / - / S] | [+ / - / S] | [+ / - / S] |
| **Total Pluses (+)** | | 0 | [Count] | [Count] | [Count] |
| **Total Minuses (-)** | | 0 | [Count] | [Count] | [Count] |
| **Weighted Score** | | 0 | [Score] | [Score] | [Score] |

* **+** = Better than baseline
* **-** = Worse than baseline
* **S** = Same as baseline
