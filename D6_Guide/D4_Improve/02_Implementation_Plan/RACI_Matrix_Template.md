# RACI Matrix Template

## English
**Project / Initiative:** [Name]

| Task / Deliverable | [Role 1, e.g., PM] | [Role 2, e.g., Sponsor] | [Role 3, e.g., IT] | [Role 4, e.g., Operations] |
|--------------------|--------------------|-------------------------|--------------------|----------------------------|
| [Task 1] | [R/A/C/I] | [R/A/C/I] | [R/A/C/I] | [R/A/C/I] |
| [Task 2] | [R/A/C/I] | [R/A/C/I] | [R/A/C/I] | [R/A/C/I] |
| [Task 3] | [R/A/C/I] | [R/A/C/I] | [R/A/C/I] | [R/A/C/I] |

* **R (Responsible):** Does the work.
* **A (Accountable):** Approves the work (Only one 'A' per task).
* **C (Consulted):** Provides input before work is done.
* **I (Informed):** Kept updated on progress.

## Arabic
# قالب مصفوفة المسؤوليات (RACI)

**المشروع / المبادرة:** [الاسم]

| المهمة / المخرج | [الدور 1، مثال: مدير المشروع] | [الدور 2، مثال: الراعي] | [الدور 3، مثال: تقنية المعلومات] | [الدور 4، مثال: العمليات] |
|-----------------|-------------------------------|-------------------------|----------------------------------|---------------------------|
| [المهمة 1] | [R/A/C/I] | [R/A/C/I] | [R/A/C/I] | [R/A/C/I] |
| [المهمة 2] | [R/A/C/I] | [R/A/C/I] | [R/A/C/I] | [R/A/C/I] |
| [المهمة 3] | [R/A/C/I] | [R/A/C/I] | [R/A/C/I] | [R/A/C/I] |

* **R (مسؤول - Responsible):** ينفذ العمل.
* **A (مساءل - Accountable):** يعتمد العمل (يجب أن يكون هناك 'A' واحد فقط لكل مهمة).
* **C (مستشار - Consulted):** يقدم المدخلات قبل تنفيذ العمل.
* **I (مُبلَّغ - Informed):** يتم إبلاغه بالتقدم.
