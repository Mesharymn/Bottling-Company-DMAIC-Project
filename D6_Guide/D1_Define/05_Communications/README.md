# D1_Define — Communications Templates

## English
This folder contains all communication templates for the Define phase. Use these as starting points and adapt them to your project's specific context.

| File | Purpose |
|------|---------|
| `EMAIL_01_Project_Kickoff.md` | Announce the project launch to all stakeholders |
| `EMAIL_02_Data_Request.md` | Request historical data from relevant teams |
| `MOM_01_Kickoff_Meeting.md` | Minutes of the project kickoff meeting |
