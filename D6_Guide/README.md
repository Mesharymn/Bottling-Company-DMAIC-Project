# DMAIC Training Plan

## English
This directory contains a comprehensive, case-agnostic training plan for all five phases of the DMAIC (Define, Measure, Analyze, Improve, Control) methodology. It is designed to be used by any team or organization looking to implement Six Sigma (6S) process improvements, independent of the specific project context.

Each subfolder contains a step-by-step training guide, essential tools and templates with free resource links, and recommended video training for that specific phase.

### Training Modules
| Phase | Folder | Description |
|-------|--------|-------------|
| **Define** | `01_Define_Training/` | Project scoping, stakeholder analysis, and Voice of the Customer (VOC). |
| **Measure** | `02_Measure_Training/` | Data collection, Measurement System Analysis (MSA), and process capability. |
| **Analyze** | `03_Analyze_Training/` | Root cause analysis, 5 Whys, and hypothesis testing. |
| **Improve** | `04_Improve_Training/` | Solution selection, pilot testing, and implementation planning. |
| **Control** | `05_Control_Training/` | Control plans, Statistical Process Control (SPC), and project handover. |

## Arabic
# خطة تدريب 6S (DMAIC)

يحتوي هذا الدليل على خطة تدريب شاملة وعامة لجميع المراحل الخمس لمنهجية 6S (التعريف، القياس، التحليل، التحسين، التحكم). تم تصميمه ليستخدمه أي فريق أو مؤسسة تتطلع إلى تنفيذ تحسينات عمليات 6S، بشكل مستقل عن سياق المشروع المحدد.

يحتوي كل مجلد فرعي على دليل تدريب خطوة بخطوة، والأدوات والقوالب الأساسية مع روابط موارد مجانية، وتدريب فيديو موصى به لتلك المرحلة المحددة.

### وحدات التدريب
| المرحلة | المجلد | الوصف |
|---------|--------|-------|
| **التعريف** | `01_Define_Training/` | تحديد نطاق المشروع، وتحليل أصحاب المصلحة، وصوت العميل (VOC). |
| **القياس** | `02_Measure_Training/` | جمع البيانات، وتحليل نظام القياس (MSA)، وقدرة العملية. |
| **التحليل** | `03_Analyze_Training/` | تحليل الأسباب الجذرية، ولماذا الخمس، واختبار الفرضيات. |
| **التحسين** | `04_Improve_Training/` | اختيار الحلول، والتجربة التجريبية، وتخطيط التنفيذ. |
| **التحكم** | `05_Control_Training/` | خطط التحكم، والتحكم الإحصائي في العمليات (SPC)، وتسليم المشروع. |
