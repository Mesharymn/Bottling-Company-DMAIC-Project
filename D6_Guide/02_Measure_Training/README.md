# Training Plan — Measure Phase

## English
This folder contains a case-agnostic, step-by-step training guide for the DMAIC **Measure** phase. It focuses on establishing a reliable baseline for the current process performance before any improvements are made.

### Step-by-Step Training Guide
1. **Map the Current State Process:** Create a detailed As-Is process map to understand the exact workflow.
2. **Develop a Data Collection Plan:** Decide what to measure, how to measure it, and sample size.
3. **Perform Measurement System Analysis (MSA):** Conduct a Gage R&R study to ensure the data collection method is accurate and repeatable.
4. **Collect Baseline Data:** Gather historical or real-time data based on the collection plan.
5. **Calculate Process Capability:** Determine the current Sigma level, Yield, DPMO, Cp, and Cpk.

## Arabic
# خطة التدريب — مرحلة القياس

يحتوي هذا المجلد على دليل تدريبي تدريجي عام يصلح لأي حالة دراسية لمرحلة **القياس** في منهجية 6S. يركز على إنشاء خط أساس موثوق لأداء العملية الحالي قبل إجراء أي تحسينات.

### دليل التدريب خطوة بخطوة
1. **رسم خريطة حالة العملية الحالية:** أنشئ خريطة عملية تفصيلية للوضع الراهن لفهم سير العمل الدقيق.
2. **تطوير خطة جمع البيانات:** قرر ما يجب قياسه، وكيفية قياسه، وحجم العينة.
3. **إجراء تحليل نظام القياس (MSA):** قم بإجراء دراسة التكرارية والتوافقية (Gage R&R) لضمان دقة طريقة جمع البيانات وقابليتها للتكرار.
4. **جمع بيانات خط الأساس:** اجمع البيانات التاريخية أو في الوقت الفعلي بناءً على خطة الجمع.
5. **حساب قدرة العملية:** حدد مستوى 6S الحالي، والعائد، والعيوب لكل مليون فرصة (DPMO)، و Cp، و Cpk.

### الأدوات والقوالب الأساسية
| الأداة | الغرض | رابط المورد المجاني |
|--------|--------|---------------------|
| **خطة جمع البيانات** | هيكلة عملية جمع البيانات | [خطة بيانات GoLeanSixSigma](https://goleansixsigma.com/data-collection-plan/) |
| **قالب دراسة التكرارية والتوافقية** | التحقق من صحة نظام القياس | [دليل ASQ لدراسة التكرارية والتوافقية](https://asq.org/quality-resources/gage-rr) |
| **حاسبة قدرة العملية** | حساب Cp و Cpk ومستوى 6S | [حاسبة قدرة iSixSigma](https://www.isixsigma.com/tools-templates/capability-indices/) |
| **خريطة تدفق القيمة (الحالية)** | رسم خرائط تفصيلية للعملية | [قوالب Lucidchart لخريطة تدفق القيمة](https://www.lucidchart.com/pages/examples/value-stream-mapping) |
