# D4 — Improve Phase
## Water Bottling Company | DMAIC Project

> The Improve phase answers one question: **"How do we fix the root causes and verify the solutions?"**

---

## Improvement Plan for Root Causes

Based on the Analyze phase findings, we are deploying the following solutions to address the confirmed root causes:

| Root Cause | Proposed Solution | Expected Impact | ROI Strategy |
|------------|-------------------|-----------------|--------------|
| **RC-01:** Outdated Maintenance Plan | Update PM plan + implement TPM | 60% reduction in leakage | Calculate downtime savings vs TPM cost |
| **RC-02:** No Formal PM Program | Deploy formal PM schedule | 40% reduction in downtime | Monitor MTBF improvement |
| **RC-03:** No SMED Procedure | Implement SMED for all SKU changes | Changeover time ≤ 20 mins | Value of regained production capacity |
| **RC-04:** Worn Torque Sensor | Replace + calibrate torque sensor | Eliminate cap deformation | Cost of scrap reduction |
| **RC-05:** No CI Culture | Launch Kaizen program + Suggestion system | Sustain improvements | Track implemented suggestions |
| **RC-06:** Push Production | Implement Kanban pull system | Eliminate overproduction waste | Reduce WIP inventory costs |

---

## Behavior Design & Change Management (ADKAR & COM-B)

To ensure the technical solutions succeed, we must design for the human element:
- **Capability:** Train operators on the new SMED and PM procedures.
- **Opportunity:** Provide the right tools and visual cues (Poka-Yoke) to make the new process the default.
- **Motivation:** Use ADKAR to build awareness and desire for the change before training begins.

## خطة التحسين للأسباب الجذرية

بناءً على نتائج مرحلة التحليل، سنقوم بتطبيق الحلول التالية لمعالجة الأسباب الجذرية المؤكدة:

| السبب الجذري | الحل المقترح | التأثير المتوقع | استراتيجية العائد على الاستثمار (العائد على الاستثمار) |
|-------------|------------|----------------|-------------------------------------|
| **السبب الجذري-01:** خطة الصيانة قديمة | تحديث خطة الصيانة الوقائية + تنفيذ الصيانة الإنتاجية الشاملة | تخفيض التسرب بنسبة 60% | حساب توفير التوقف مقابل تكلفة الصيانة الإنتاجية الشاملة |
| **السبب الجذري-02:** لا يوجد برنامج الصيانة الوقائية | تطبيق جدول صيانة وقائية رسمي | تخفيض التوقف بنسبة 40% | مراقبة تحسن متوسط الوقت بين الأعطال (متوسط الوقت بين الأعطال) |
| **السبب الجذري-03:** لا يوجد إجراء تقليل وقت تغيير القوالب | تنفيذ تقليل وقت تغيير القوالب لجميع تحويلات وحدات المنتج | وقت التحويل ≤ 20 دقيقة | قيمة الطاقة الإنتاجية المستردة |
| **السبب الجذري-04:** مستشعر عزم متآكل | استبدال ومعايرة مستشعر العزم | القضاء على تشوه الغطاء | تقليل تكلفة الخردة (Scrap) |
| **السبب الجذري-05:** لا توجد ثقافة التحسين المستمر | إطلاق برنامج التحسين المستمر ونظام اقتراحات | الحفاظ على التحسينات | تتبع الاقتراحات المنفذة وقيمتها |
| **السبب الجذري-06:** إنتاج بالدفع (Push) | تنفيذ نظام سحب السحب البصري | القضاء على هدر الإنتاج الزائد | تقليل تكاليف المخزون قيد التشغيل (المخزون قيد التشغيل) |

## تصميم السلوك وإدارة التغيير (الوعي والرغبة والمعرفة والقدرة والتعزيز و القدرة والفرصة والدافع)

لضمان نجاح الحلول التقنية، يجب أن نصمم للعنصر البشري:
- **القدرة (Capability):** تدريب المشغلين على إجراءات تقليل وقت تغيير القوالب والصيانة الوقائية الجديدة.
- **الفرصة (Opportunity):** توفير الأدوات المناسبة والإشارات البصرية (منع الأخطاء) لجعل العملية الجديدة هي الخيار الافتراضي.
- **الدافع (Motivation):** استخدام نموذج الوعي والرغبة والمعرفة والقدرة والتعزيز لبناء الوعي والرغبة في التغيير قبل بدء التدريب.

---

## قائمة التحقق لمرحلة التحسين

- [ ] عصف ذهني للحلول لجميع الأسباب الجذرية المؤكدة
- [ ] ترتيب أولويات الحلول باستخدام مصفوفة التأثير/الجهد
- [ ] تصميم وتنفيذ اختبارات تجريبية (التجربة التجريبية) للحلول عالية المخاطر
- [ ] التحقق من نتائج الاختبار التجريبي إحصائياً
- [ ] اعتماد خطة التطبيق التفصيلية (الجدول الزمني + المسؤول والمساءل والمستشار والمبلغ)
- [ ] تطوير خطة إدارة التغيير (الوعي والرغبة والمعرفة والقدرة والتعزيز)
- [ ] تطبيق الحلول بالكامل على خط الإنتاج
- [ ] اجتياز مراجعة بوابة مرحلة التحسين (Tollgate)
