# 02 — Implementation Plan
## Water Bottling Company | Improve Phase

> **Purpose:** To define exactly WHO does WHAT and WHEN to ensure the selected solutions are deployed successfully without disrupting ongoing production.

---

## RACI Matrix

The RACI matrix ensures clear accountability for each solution deployment:
- **R**esponsible: Does the work
- **A**ccountable: Owns the result (Only one 'A' per task)
- **C**onsulted: Provides input before action
- **I**nformed: Kept updated on progress

| Solution | Plant Manager | Maintenance Lead | Quality Lead | Production Sup. | Operators |
|----------|---------------|------------------|--------------|-----------------|-----------|
| **TPM Rollout** | A | R | C | C | I |
| **Smart Sensor** | I | R/A | C | I | I |
| **SMED Implementation**| A | C | I | R | C |
| **Kanban System** | A | I | I | R | C |
| **Kaizen Huddles** | I | I | I | A | R |

---

## High-Level Gantt Chart (4-Week Rollout)

| Week | Phase | Key Activities |
|------|-------|----------------|
| **Week 1** | Preparation | Procure smart sensor, design Kanban cards, map external SMED tasks |
| **Week 2** | Training | Train operators on Kaizen, train setup crew on SMED |
| **Week 3** | Deployment | Install sensor, launch Kaizen huddles, start TPM scheduling |
| **Week 4** | Validation | Monitor MTBF, measure changeover times, adjust Kanban limits |

---

## مصفوفة RACI

تضمن مصفوفة RACI وضوح المساءلة لكل عملية نشر حل:
- **R (مسؤول):** يقوم بالعمل الفعلي
- **A (مساءل):** يملك النتيجة (يجب أن يكون هناك 'A' واحد فقط لكل مهمة)
- **C (مُستشار):** يقدم مدخلات قبل اتخاذ الإجراء
- **I (مُبلغ):** يتم إبقاؤه على اطلاع بالتقدم

| الحل | مدير المصنع | قائد الصيانة | قائد الجودة | مشرف الإنتاج | المشغلون |
|------|-------------|--------------|-------------|--------------|----------|
| **إطلاق TPM** | A | R | C | C | I |
| **المستشعر الذكي**| I | R/A | C | I | I |
| **تطبيق SMED** | A | C | I | R | C |
| **نظام Kanban** | A | I | I | R | C |
| **اجتماعات Kaizen**| I | I | I | A | R |
